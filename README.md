# utsho.space

My personal site — live at [utsho.space](https://utsho.space).

A static site built with [Astro](https://astro.build): big type, a warm paper
background, one accent color, and light scroll animations that switch off if you
prefer reduced motion. Hardly any JavaScript, so it loads fast and hosts anywhere.

## Running it

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static build in dist/
```

Text and links live in `src/config.ts`. Colors and fonts are CSS variables at
the top of `src/styles/global.css`.
